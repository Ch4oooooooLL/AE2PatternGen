# i18n Support Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add bilingual localization support for core player-facing texts by replacing hardcoded strings with lang keys and a shared translation utility.

**Architecture:** Introduce a small `I18nUtil` abstraction over Minecraft localization APIs. GUI code reads localized strings through this utility, while server-side player messages and command outputs use translation components where suitable. All keys are centralized in `en_US.lang` and `zh_CN.lang`.

**Tech Stack:** Java 8, Minecraft Forge 1.7.10 APIs (`StatCollector`, `ChatComponentTranslation`), Gradle (RFG/GTNH)
